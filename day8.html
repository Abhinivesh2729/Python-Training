Python List Operations Day 8

1. Creating Lists

# Empty list
empty_list = []

# List with items
numbers = [1, 2, 3, 4, 5]
mixed_list = [1, "hello", 3.14, True]

# Creating a list using the list() constructor
chars = list("hello") # ['h', 'e', 'l', 'l', 'o']
Lists can contain items of different data types (integers, strings, floats, booleans, etc.) and even other lists.

2. Accessing List Elements

fruits = ["apple", "banana", "cherry", "date", "elderberry"]

# Accessing by index (starts at 0)
first_fruit = fruits[0] # "apple"
third_fruit = fruits[2] # "cherry"

# Negative indices count from the end
last_fruit = fruits[-1] # "elderberry"
second_last = fruits[-2] # "date"

# Slicing (getting a portion of the list)
first_three = fruits[0:3] # ["apple", "banana", "cherry"]
# OR
first_three = fruits[:3] # Same result

# From index 2 to the end
remaining = fruits[2:] # ["cherry", "date", "elderberry"]

3. Modifying Lists

colors = ["red", "green", "blue"]

# Changing an element
colors[0] = "yellow" # Now: ["yellow", "green", "blue"]

# Adding elements
colors.append("purple") # Add to end: ["yellow", "green", "blue", "purple"]
colors.insert(1, "orange") # Insert at position: ["yellow", "orange", "green", "blue", "purple"]

# Combining lists
more_colors = ["pink", "black"]
colors.extend(more_colors) # ["yellow", "orange", "green", "blue", "purple", "pink", "black"]
# OR
combined = colors + more_colors # Creates a new list

4. Removing Elements

animals = ["dog", "cat", "rabbit", "elephant", "cat"]

# Remove first occurrence by value
animals.remove("cat") # ["dog", "rabbit", "elephant", "cat"]

# Remove by index and return the value
removed = animals.pop(1) # removed = "rabbit", animals = ["dog", "elephant", "cat"]
last = animals.pop() # No index means remove last item: last = "cat", animals = ["dog", "elephant"]

# Delete an element or slice
del animals[0] # animals = ["elephant"]

# Clear all elements
animals.clear() # animals = []

5. List Operations

numbers = [1, 5, 2, 8, 3]

# Length
length = len(numbers) # 5

# Sorting
numbers.sort() # Modifies original: [1, 2, 3, 5, 8]
# OR
sorted_numbers = sorted(numbers) # Creates new list, original unchanged

# Reversing
numbers.reverse() # Modifies original: [8, 5, 3, 2, 1]
# OR
reversed_list = numbers[::-1] # Creates new list

# Count occurrences
count = [1, 2, 3, 1, 1, 4].count(1) # 3

# Find index of first occurrence
index = ["a", "b", "c", "d"].index("c") # 2

# Check if item exists in list
exists = "apple" in ["banana", "apple", "cherry"] # True

6. List Comprehensions

# Create a list of squares
squares = [x**2 for x in range(5)] # [0, 1, 4, 9, 16]

# Create a filtered list
even_numbers = [x for x in range(10) if x % 2 == 0] # [0, 2, 4, 6, 8]
List comprehensions provide a concise way to create lists based on existing lists or other iterables.

7. Nested Lists

# Creating a nested list (2D list)
matrix = [
  [1, 2, 3],
  [4, 5, 6],
  [7, 8, 9]
]

# Accessing elements in nested lists
element = matrix[1][2] # Row 1, Column 2: 6

8. Common Patterns

# Finding maximum and minimum values
max_val = max([5, 9, 2, 1, 7]) # 9
min_val = min([5, 9, 2, 1, 7]) # 1

# Sum of all items
total = sum([1, 2, 3, 4]) # 10

# Looping through a list
colors = ["red", "green", "blue"]
for color in colors:
    print(color)

# Looping with index
for i, color in enumerate(colors):
    print(f"Index {i}: {color}")
